Android开发-上传和更新项目到GitHub

Jul 06 2019

GitHub是一个面向开源及私有软件项目的托管平台，在团队项目开发中可以有效地进行代码的托管和团队协作分工，本文详细记录在Android studio下如何进行代码的上传以及更新。



1.Git配置

在Android studio的设置中找到Git配置栏，在地址栏中输入Git的安装路径，然后点击Test，若展示Git版本则表示配置完成。

2.GitHub登录

配置完Git别急着关闭设置窗口，在Git的配置的上方既有GitHub的配置，点击并输入你的GitHub账号密码以登录。

3.上传项目到GitHub

在菜单栏中有一个VCS，选择里面的Import into Version Control的Share Project on GitHub，跳出如下窗口。

填好项目名称和项目描述就可以上传了，若不想项目开源，勾选Private创建私有项目。

项目上传后登录你的GitHub就可以看见刚刚上传的项目了。🙈

4.项目更新

往往一个项目不是一蹴而就的，需要进行不断的迭代更新，同样选择VCS下的Git中的Commit File，跳出如下窗口。

Commit Message是有关这次代码更新描述，右侧是一些配置选项，配置好后点击Commit即可。

注意！跳出如上警告信息，忽略它点击Commit。

随后再次选择Git，选择Push。

点击右侧的文件可以快速查看此次更新和旧内容的对比，确认后点击Push，等待项目上传成功。

更新项目后回到GitHub上可以看见代码已经被修改了，并且有关于更新的描述。😁
